 ⭐ Modelagem Star Schema com Python e Pandas

📌 Sobre o Projeto

Este projeto tem como objetivo demonstrar, na prática, a construção de um modelo dimensional no formato **Star Schema** utilizando Python e pandas.

A partir de uma tabela única contendo informações de vendas, foram criadas:

- Dimensões
- Chaves surrogate
- Tabela fato
- Relacionamentos lógicos

O projeto simula uma etapa comum em processos de:

- ETL
- Data Warehousing
- BI
- Engenharia de Dados

---

# 🛠️ Tecnologias Utilizadas

- Python
- Pandas
- OpenPyXL
- Jupyter Notebook
- VS Code
